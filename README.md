# 📖 Story Planer

Eine Web-App zum Planen und Verwalten von Geschichten, Romanen und anderen kreativen Werken. 

## ✨ Features

- **Bereiche** - Charaktere, Worldbuilder, Kapitel, Themen, Mechaniken; neue Bereiche können hinzugefügt werden
- **Seiten** - Titel, Inhalt, Bilder pro Seite
- **✍️ Textformatierung** - Die Texte der einzelnen Seiten können durch einfach Menüführung kursiv, fett, unterstrichen oder als Liste dargestellt werden
- **📸 Bilder** - Referenzbilder hochladen, Doppelklick für Lightbox
- **🔗 Wiki-Links** - `[[Seitenname]]` im Text, Klick öffnet andere Seite
- **🔄 Auto-Close** - Fenster schließt automatisch nach Speichern
- **💾 LocalStorage** - Alle Daten bleiben im Browser gespeichert
- **📤 Export/Import** - Sichern und Wiederherstellen als JSON, zur Sicherung und Nutzung zwischen verschiedenen Geräten
- **🗑️ Reset** - Alle Daten löschen mit Neustart
- **🖱️ Klick auf Thumbnail** - Seite öffnen
- **🤚 Drag & Drop** - Seiten können per Drag & Drop in andere Bereiche gezogen und innerhalb eines Bereichs neu angeordnet werden
- **📱 Responsive** — Funktioniert auf Desktop und Mobilgeräten, größere Buttons auf Touch-Geräten
